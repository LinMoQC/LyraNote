# 知识图谱

LyraNote 自动从你上传的内容中构建**知识图谱**，提取实体和关系，帮助你发现整个知识库中隐藏的非显著关联。

## 工作原理

当你上传或创建文档时，LyraNote 的 AI 流水线会：

1. **提取实体** — 人物、组织、技术、概念及其他命名实体
2. **识别关系** — 基于上下文理解实体之间的关联
3. **增量合并** — 将新实体和关系合并到现有图谱中

例如，从一组 AI 研究论文中，图谱可能自动建立以下关联：

- `LangGraph` → _被使用于_ → `深度研究 Agent`
- `pgvector` → _存储_ → `向量嵌入`
- `Transformer` → _奠定基础_ → `注意力机制`

## 探索图谱

图谱视图使用**力导向布局** — 高度连接的节点会自然聚集在一起，揭示你的知识结构。

### 交互操作

| 操作 | 结果 |
|---|---|
| 单击节点 | 查看实体详情和关联来源文档 |
| 拖拽节点 | 在布局中重新定位 |
| 滚动 | 放大 / 缩小 |
| 双击背景 | 重置视图 |

### 筛选

使用筛选面板：
- 仅显示特定类型的实体（如仅显示"技术"或"人物"节点）
- 按名称搜索特定实体
- 将图谱限制在某个日期范围内

## 笔记也是知识来源

LyraNote 不仅索引你导入的来源文件，还会索引**你自己写的笔记**。这意味着：

- 你在笔记本 A 中写的研究总结，在笔记本 B 的对话中也可以被 AI 引用
- 全局对话可以检索到你的个人研究笔记内容
- 你的思考和分析成果成为知识网络的一部分，而不仅仅是导入的来源

这形成了完整的**知识闭环**：来源 → AI 分析 → 笔记 → 知识库 → 未来 AI 回答。

## 使用场景

- **研究** — 查看阅读笔记中的概念如何相互关联
- **项目规划** — 可视化人员、工具和决策之间的依赖关系
- **学习** — 梳理某一学科领域中各主题的关联与构建方式
- **综合** — 发现不同研究线索之间的意外关联
