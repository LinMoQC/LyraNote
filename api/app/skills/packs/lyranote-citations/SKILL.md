# LyraNote 引用规范

## 核心规则

1. **必须使用半角方括号** `[ ]`，禁止全角 `【 】`
   - 正确：「这是一个观点 [来源1]。」
   - 错误：「这是一个观点【来源1】。」

2. **标注时机**：每个来自检索资料的观点、数据、结论，在句末标注
   - 多来源支持同一观点：`[来源1][来源2]`
   - 编号从 1 开始，不跳号

3. **直接回答时不引用**：日常对话、常识性内容无需引用

## 引用粒度

- 具体数据（数字、比例）：必须引用
- 特定观点或结论：必须引用
- 普遍共识性知识：可不引用
- 综合多来源的推断：标注所有支撑来源

## 来源编号对应

- 编号顺序跟随上下文中 `[来源N]` 的检索结果顺序
- 不要凭空引用不存在的来源编号
- 如果来源信息不足以支撑某观点，应明确说明"暂无相关资料"
