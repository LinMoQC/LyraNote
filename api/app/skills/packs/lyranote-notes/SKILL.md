# LyraNote 笔记创建规范

使用 `create_note_draft` 工具创建笔记时，遵循以下规范。

## 笔记结构模板

```markdown
# [主题名称]

## 概述
一段话说明这篇笔记的主要内容和目的。

## 核心内容
（主体内容，按逻辑分节）

## 关键要点
- 要点 1
- 要点 2
- 要点 3

## 参考来源
- 来源 1：[标题]
- 来源 2：[标题]
```

## 命名规范

- 标题简洁具体，避免模糊词（如"关于XX的笔记"→改为"XX 核心机制解析"）
- 使用名词短语，不超过 20 字
- 研究类：「[主题] 研究综述」
- 分析类：「[主题] 分析报告」
- 概念类：「[概念名] 详解」

## 内容质量要求

1. 每篇笔记聚焦**一个主题**，不要混杂多个话题
2. 引用来源时注明出处，格式：> 来源：《文档标题》
3. 重要术语首次出现时给出简短定义
4. 结尾提供 2-3 个后续研究方向或相关问题

## 何时主动创建笔记

- 用户完成了一次深度研究对话后
- 对话产生了有价值的综合性分析
- 用户说"帮我整理一下"、"记录下来"等
